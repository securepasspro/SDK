# SecurePass Pro JavaScript SDK

## Overview
This is the official JavaScript SDK for SecurePass Pro - a secure password generator with cryptographically secure 256-bit encryption. The SDK provides a simple interface for generating secure passwords and managing teams through the SecurePass Pro API.

## Project Status
**Current State**: Fully set up and running in Replit environment
- ✅ Dependencies installed
- ✅ SDK built with Rollup
- ✅ Documentation server running on port 5000
- ✅ Examples and documentation accessible

1. Builds the SDK using Rollup
2. Starts an HTTP server on port 5000
3. Serves documentation and examples

## SDK Usage

### Basic Example
```javascript
import SecurePassSDK from '@securepasspro/sdk';

const sdk = new SecurePassSDK('spro_your-api-key-here');

// Generate a password
const result = await sdk.generatePassword({ length: 16 });
console.log(result.password);
```

### API Methods
- `generatePassword(options)` - Generate a single password
- `generateBulkPasswords(count, options)` - Generate multiple passwords
- `getTeamInfo(teamId)` - Get team information
- `addTeamMember(teamId, email, role)` - Add team member
- `removeTeamMember(teamId, email)` - Remove team member
- `updateTeamMemberRole(teamId, email, role)` - Update member role
- `getUsage()` - Get usage statistics
- `testConnection()` - Test API connection

## Important Notes

### For Development
- The built files in `dist/` are generated by Rollup and should not be edited directly
- All source changes should be made in `src/securepasspro.js`
- After changes, run `npm run build` to rebuild
- The documentation server automatically serves the latest built files

### Security
- API keys must start with `spro_` prefix
- Never expose API keys in client-side code
- SDK validates all inputs before sending to API
- Implements request timeout (10 seconds)
- No password storage or logging

### Browser Compatibility
- Modern browsers with ES6 support
- UMD build available for older browsers
- CDN distribution available via jsDelivr
